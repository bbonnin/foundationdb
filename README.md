FoundationDB
============

Various FoundationDB examples.

* [FoundationDB] (http://foundationdb.com/), [NodeJS API] (http://foundationdb.com/documentation/beta2/api-node.html)
* [Node.js](http://nodejs.org/), [Express](http://expressjs.com/) (REST API)

## Installation

``` bash
     npm install
```

## Run

* Start NodeJS

``` bash
     node app
```
